Image Member::uploadImg ( const std::string &  filename,
JNIEnv *  env = manager->getEnv() 
)

上传本地图片，务必要用绝对路径 由于mirai要区分图片发送对象，所以使用本函数上传的图片只能发到群

注意
最大支持图片大小为30MB
异常
-可能抛出UploadException异常代表路径无效或大小大于30MB-可能抛出MemberException找不到群或群成员
